Cloud QC — Launched 2024

SeisCloudNAV

Real-time navigation quality control in your browser.

Traditional QC is backwards-looking. By the time the report arrives, the vessel has moved on and the budget for remediation is already spent. SeisCloudNAV changes the model entirely.

Our proprietary compression algorithm transmits navigation data from the field to cloud dashboards within minutes of acquisition — without any loss of fidelity. E&P companies, their contractors, and independent consultants can all monitor the same survey data simultaneously.

  • Live navigation data in your browser — minutes after acquisition
  • No client-side software installation required
  • Real-time integrity and format checking
  • User-configurable plots and acceptance thresholds
  • Unlimited user logins per project
  • Survey-wide trend analysis — spot issues before they compound
  • Currently supporting P1 format — OBS in active development

Independent Verification

First Line QC (FLQC)

The independent audit your survey depends on.

FLQC is FGPS's comprehensive navigation data verification service. Within 24 hours of receiving your dataset, our team produces a structured QC report covering every aspect of navigation parameter validity, data completeness, and deliverable compliance.

Because FGPS is entirely independent — not owned by any contractor or parent company — the FLQC report carries genuine authority. Contractors cannot argue with independent verification. E&P clients can rely on it for sign-off.

  • 24-hour turnaround from data receipt
  • Comprehensive navigation parameter verification
  • Format compliance checking for P1/11 and UKOOA standards
  • Fully independent — not affiliated with any contractor
  • Structured report ready for E&P client acceptance

Data Services

Remote Navigation
Data Processing

Full P2 to P1 processing, handled by FGPS specialists.

When you need processing done to the highest standard — but your team's capacity is committed elsewhere — FGPS can process your navigation data remotely. Whether a complex multi-vessel 3D survey or legacy data reprocessing, our team applies the same tools and techniques used by the world's leading contractors.

All processing uses the full SeisPos suite, with comprehensive quality control, reporting, and communication throughout. Output is delivered in your required format with complete documentation.

  • Full P2-to-P1 processing by FGPS engineers
  • Handles 2D, 3D, OBC, and OBN survey geometries
  • Legacy data reprocessing and format conversion
  • Detailed reporting at every processing stage
  • Deliverables in IOGP P1/11, UKOOA P1/90, or custom formats

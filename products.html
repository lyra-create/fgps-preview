Which Product Do I Need?

Quick Guide

Not sure where to start? Here's the simplest way to think about our software suite.

Processing

I need to process raw navigation data

Start with SeisPos for P2-to-P1 processing. Add FBComp if you're working with OBC/OBN data.

QC

I need to quality control final data

Start with P1Tools for comprehensive P1 QC. Use SPSPro for SPS-format data.

Planning

I need to plan a survey or analyse coverage

Use SeisPlan for preplots and SeisBin for 3D coverage and infill analysis.

Utilities

I need to convert, check, or inspect data

Use Px11Utils for P1/11 and P2/11 formats, SurvUtil for geodetic conversions, SEGYTools for SEG-Y.

SeisPos

The Complete Processing Engine

Fully comprehensive processing and QC package for marine 2D and 3D seismic navigation data. From raw P2 positioning data to final coordinates — SeisPos handles the entire chain.

  • Supports UKOOA P2/P1, IOGP P2/11, P1/11, and SPS formats
  • All marine seismic survey types — streamer, OBC, OBN
  • Proprietary algorithms for optimum performance at low cost
  • Comprehensive quality control and reporting
  • Handles the most demanding datasets

P1Tools

Deep QC & Analysis

The tool for surveyors who need to do more than "accept/reject". Comprehensive quality appraisal of final marine 2D and 3D positioning data with trend analysis, statistical testing, and audit-ready reporting.

  • QC Offsets — compare any node positions with user-specified config
  • QC Nodes — shot-to-shot movement, depth, streamer analysis
  • Compare — diff two P1 datasets with visual analysis
  • Trend Analysis — survey-wide statistical testing with automated flagging
  • Replay — interactive graphical replay of P1 data
  • Streamer Shape visualisation
  • Full utilities: Extract, Coordinate Conversions, Preplot, DXF, Depth Corrections
Download Datasheet ↗

SeisBin

3D Coverage Intelligence

Comprehensive 3D coverage visualisation, analysis, and reporting. Multiple offset groups and sampling parameter sets make switching between different views a button-click away.

  • Powerful infill analysis — automated and interactive
  • Full Fold and Null Fold perimeters
  • Manual and ADS-file based editing
  • Comprehensive QC, coverage, and infill reporting
Download Datasheet ↗

SeisPlan

Survey Planning, Done Right

Create preplots for all types of marine seismic surveys. Designed for multi-user environments with an intuitive interface that makes complex survey geometries manageable.

  • All marine seismic survey types supported
  • Multi-user support for collaborative planning
  • Interactive visual editing
  • EPSG-based geodetic definitions
  • Comprehensive output formats
Download Datasheet ↗

SPSPro

Land & OBC Specialist

Interactive tool for analysing and checking all aspects of land and OBC positioning data in SPS format. Record integrity, position mapping, active spread visualisation, and receiver coordinate computation.

  • Record integrity checking
  • Source and receiver position mapping by type and index
  • Interactive mapping of active spread
  • Acquisition replay
  • Receiver coordinate computation using acoustics or first breaks
Download Datasheet ↗

FBComp

OBS Receiver Positioning

Compute receiver group coordinates using any combination of acoustic ranges, bearings, USBL X and Y, constrained receiver positions, and first breaks. Essential for the growing OBS market.

  • Acoustic ranges and bearings
  • USBL positioning data
  • Constrained receiver positions (GPS)
  • First break picks
  • Any combination of the above
Download Datasheet ↗

SEGYTools

SEG-Y QC & First Break Picking

Quality control and first break picking for SEG-Y format data. View headers, plot traces with filtering and AGC, pick first breaks, and export results.

  • Text, binary, and trace header viewing and editing
  • Standard trace display with AGC, LMO, bandpass filters
  • First break pick parameterisation and display
  • Trace sorting by up to 10 header fields
  • Export to SPS and pick files
Download Datasheet ↗

Utilities Suite

SurvUtil, Px11Utils, AppSync, Databases

Geodetic conversions (EPSG-based), P2/11 and P1/11 format checking and conversion, project management, and shared databases. Px11Utils Lite is available free for evaluation.

  • SurvUtil — EPSG-based geodetic and survey utilities
  • Px11Utils — Format checking, conversion, and editing for Px/11 formats
  • AppSync — Unified project management shell
  • EPSG, geoid, magnetic model, and map databases
